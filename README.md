## 프로젝트 소개
* 크롬 브라우저 확장 프로그램에 기반한 프로젝트 이며, XPath 추출을 위한 도구입니다.
  * 크롤링 기능 구성시 RPA 기능에 활용이 가능한 크롬 브라우저의 웹 확장 도구
  * 관련 의문사항이나 로직 수정은 [이슈 생성](#)을 통하여 문의 및 진행하시길 바랍니다.
